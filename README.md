# Gridly

**Know Before You Go.**

Gridly is a real-time mobility and hazard intelligence platform built to help people understand what is happening around them before they travel.

This pilot release currently focuses on Liberty County, Texas, using public FRA railroad crossing data and community-submitted live reports.

## Current Version

**Gridly V6.0 WOW Dark Launch**

## Pilot Coverage

Liberty County, Texas

## Core Features

- Live map interface
- Public FRA railroad crossing data
- Community blocked / cleared reports
- Supabase shared report storage
- Active report summary
- Use My Location
- Crossing visibility categories
- Local crossing override support
- Mobile-responsive dark interface
