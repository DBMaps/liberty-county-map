# Gridly Beta QR Code

## Official URL

https://dbmaps.github.io/liberty-county-map/

## Purpose

This directory registers the official Gridly Beta QR code assets for print, review, and beta-distribution references. The QR code is intended to route beta testers to the stable beta URL without temporary version parameters.

## File inventory

- `gridly-beta-qr-512.png` — 512 px PNG QR asset for small digital placements.
- `gridly-beta-qr-1024.png` — 1024 px PNG QR asset for standard digital and lightweight print placements.
- `gridly-beta-qr-2048.png` — 2048 px PNG QR asset for high-resolution print or source-of-truth distribution needs.
- `README.md` — registration notes, usage guidance, and validation checklist.

## Usage guidance

Use these QR assets only when the intended destination is the official Gridly Beta URL above. Do not relabel these assets for temporary, experimental, or environment-specific links.

Do not use temporary version parameters, including `beta-clean` version parameters, in QR destinations or campaign materials that reference these registered assets.

Mobile Portrait is the supported beta experience. Desktop remains gated. Landscape remains gated.

## QR validation checklist

Before publishing or distributing the QR code, complete the following human validation steps:

- Scan with iPhone camera.
- Scan with Android camera.
- Confirm it opens the official URL: https://dbmaps.github.io/liberty-county-map/.
- Confirm it does not include beta-clean version parameters.
