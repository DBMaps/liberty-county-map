# GRIDLY Dashboard Specification

## Purpose

The Dashboard is the home screen of the Gridly Developer Toolkit.

Every time it opens, it should immediately answer:

**Is Gridly healthy?**

---

## Repository

- Project
- Branch
- Git Status
- Last Commit

---

## Production

- Runtime Validation
- Smoke Test
- Package Validation
- Production Readiness

---

## Data Platform

- Community Packages
- Crossing Packages
- Production Crossings
- Supported Counties

---

## Documentation

- Certifications
- Release Notes
- Handoffs

---

## Toolkit

- Runtime Validator
- Smoke Test
- Inventory
- Certification
- Release
- Handoff

---

## Overall Status

Display one result only:

✅ READY TO SHIP

or

❌ BLOCKED

Every failure must include the reason.
