# Gridly Beta Install Guide

## Open The Beta Link

Open the Gridly beta link on your phone or computer. For the best mobile experience, use Chrome on Android or Safari on iPhone.

## Android Chrome

1. Open the Gridly beta link in Chrome.
2. Tap the three-dot menu.
3. Tap **Add to Home screen** or **Install app** if shown.
4. Confirm the name and tap **Add** or **Install**.
5. Reopen Gridly from the new home screen icon.

## iPhone Safari

1. Open the Gridly beta link in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Confirm the name and tap **Add**.
5. Reopen Gridly from the new home screen icon.

## Reopen Gridly Later

Use the home screen icon if you installed it. If you did not install it, reopen the same beta link in your browser.

## Replay The Walkthrough

Open **Settings**, go to **About & Support**, and choose **Show walkthrough again**.
